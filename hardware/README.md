#Hardware design and circuit diagrams.

HARDWARE COMPONENTS

1. ESP32 / Arduino Uno
2. Microphone Sensor Module
3. Buzzer
4. LED Indicator
5. Breadboard
6. Jumper Wires
7. USB Power Supply

Purpose:
These components are used to capture acoustic signals, process them, and generate alerts when abnormal sounds are detected.

CIRCUIT DESCRIPTION

# Circuit Description

The microphone sensor is connected to the ESP32/Arduino to capture sound signals. The microcontroller processes the audio data and activates an LED or buzzer when abnormal sound patterns are detected.
