# Expected Output

The system continuously monitors sound signals and identifies abnormal acoustic patterns.

Output:
- Normal Sound Detected
OR
- Pest Activity Detected
OR
- Structural Anomaly Detected

When an anomaly is detected, an alert is generated using an LED or buzzer.
